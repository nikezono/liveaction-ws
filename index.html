前置きby中園

とりあえずこのワークショップのねらいは、動画を撮って、編集しているときの、「なんとなく、こうつなぐ」(#1)というカットの感覚を養うものです。これはたくさん撮ってたくさん編集するか、たくさん映像を見るかしないと身に付かないです。僕はまだ身に付いてないです。どう撮って、どうつなげば「視聴者にとって不自然でないか」というところは、今年のスターティングのように音楽を主体にしたPVを作るだけでは身に付きづらい（と僕は思います）。PVでは「視聴者にショックを与える」ことは技術的に取り込まれてますが、映画では、うまく使わないと、そのカットのショックが「素人だな」という印象に直結するからです。

基礎知識とか

映像をやる上での基本的な知識と用語集です。とりあえずコレは頭に入っている、言わずとも意識している、意味が通る、というぐらいのスタッフの練度があると撮影がはかどります。

用語集と基礎知識は書籍とネットからあさって来たマトメにすぎないので、もっと言語として普及していないノウハウは先輩に聞いてください。

用語集

  • アイリス : レンズの絞りのこと。
  • アクションつなぎ : 人の動きでカットをつなぐこと。確実な視線誘導を行い、カットごとのショックを少なくするために、可能なかぎり使いたいモノ。
  • アシスタントディレクター : 雑用の英称ではない。ディレクターの補佐。
  • アスペクト比 : 画面の縦横のサイズ比。世界的に4:3から16:9への移行が進むが、16:10などのサイズもあり、また映画館のスクリーンサイズはシネマスコープといい12:5である。さらに、DVD-Videoの共通規格は720*480(3:2)であり、この違いによる事故はどれだけMOVEにいてもなくならない。
  • アップ : クローズアップ、アップショット、アップロードの略。「○○のアップ〜」と、「○○にフォーカスして〜」の違いを理解するとよい
  • アドリブ : 即興のこと。賛否両論の分かれる演出手法。
  • アフレコ : After Recordingの略称。映像に当てる形で音を録る手法で、リップシンクの問題上セリフ録りには適さない。
  • 粗編集（アラヘンシュウ） : ラッシュともいう。雑でもいいので、とりあえずOKテイクのカットをコンテ通りの順番に並べてみること。編集の最初に行う作業。
  • アングル : 角度のこと。カメラの位置ではなく、どちらを向いているか、のこと。高さも含めてXYZの軸のこと。だが、「そのアングルいいねー」と言う場合は「位置」も含まれることもある。
  • アンチエイリアス : CGにおいて、ベクターイメージをラスターイメージに変換した際の境界線のドット感、ギザギザをぼかす処理。
  • 板付き : シーンの冒頭から画面上に登場人物が立っていること。舞台芸能の用語だが、映像の場合も基本的にタブー。「板付きで演技をはじめさせる」ことはやってはならない。
  • イマジナリーライン : 後述するイマジナリーラインの項で説明する。想定線という意味。
  • インサートカット : インサートとも。「挿入するカット」であり、ロケーションにおける「当たり障りのないカット」のこと。たとえば、夏の日差しと太陽とか、木に止まったセミとか。カットがつながらないときに使うモノで、これをどれだけ撮ってあるか、が編集の負担を大きく変化させる。
  • インターレース処理 : 画像を上（あるいは下など特定の方向）から順に送るプログレッシブ (progressive scan) またはノンインターレース (noninterlace scan) と対になる言葉で、ラスタースキャン方式において、画像全体を飛び飛びに走査・処理する。
  • ウエストショット : 人物を撮影するときのサイズ。腰から頭まで。
  • 雲台 : 三脚の上についていて、カメラを乗せるアレ。「ヘッド」とも言う。
  • 絵コンテ : Continuityの日本語読み。絵ではなく字コンテ、写真コンテなどもある。映像の設計図。コンテは「書く」ではなく「切る」と言う。
  • SE(エスイー) : Sound Effectの略。効果音のこと。
  • NDフィルター : 光量を低下させるフィルター。
  • F値（エフち） : レンズの絞り値を示す単位。NDフィルター、F値、アイリスなどの関係については別記。
  • オーディオミキサー : いくつかの音を合成し調整する装置。複数のInputと、その調節のための簡易的なイコライザ、そして複数のOutputを持つもの。Moveには中園の私物と木村の私物がある。
  • 解像度 : 解像度（かいぞうど）とは、ビットマップ画像における画素の密度（pixel density）を示す数値である。 すなわち、画像を表現する格子の細かさを解像度と呼び、一般に1インチをいくつに分けるかによって数字で表す。
  • カチンコ : 小型の黒板やホワイトボード。カットナンバー、シーンナンバー、テイク数などを記載する。拍手木を鳴らして合図する。使い方がぶっちゃけわからない。
  • カット : カメラが回ってから止まるまでに撮影された映像。そして、切ること。
  • カット尻 : カットのアウト点の付近のこと。あるいは、使わないけど撮っておく後ろの３秒間ぐらいの間のこと。あると便利。
  • カットバック : 別々の場面を交互に写し、交互に展開していく演出手法のこと。暗示的な感情表現の演出として優秀。細かく言うと、クロスカッティングというのがそれにあたり、カットバックとは一回の動きを指すことが多い。フラッシュバックというのが、ごく短い間隔でクロスカッティングを行うことを指す。
  • 上手 : 撮影する側から見て右側。カメラの画における右側。スクリーンにおける右側。そこらへんは別記。
  • カメラ目線 : やってしまったら没になる筆頭行為の一つ。であるが、一人称視点や切り返しのショットであれば使うこともある。ファンキーモンキーベイビーズ。
  • ガンマイク : 前方に強い指向性を持ったマイクのこと。
  • 監督演出、ディレクターと同義語。脚本と同義語になりがちである。
  • 起承転結 : 序破急という言葉もある。ストーリーの骨組みのこと。これを意識しすぎた挙げ句駄作になるということもある。
  • キープ : とりあえず保留の意味。スクリプトに、「オーケー」「エヌジー」「キープ」と書いて行くと編集時にわかりやすい。
  • キーライト : ライティング（照明）のときメインとなるライト。メインライト。これと、「おさえのライト」といわれるライトによって陰影をつけたり消したりするのが「照明」の仕事。
  • 逆光 : カメラに向かって太陽が存在する位置取りのこと。基本的に、被写体の正面が陰になるので、まず避けたい状況。
  • カメリハ : カメラのリハーサルを行うこと。セットアップを行うこともあり、カメラワークの確認を行うこともある。
  • 切り返し : たとえば、二人の会話を撮るとき、交互に互いの顔を撮影すること。逆の角度から撮影するカット。便利だが、イマジナリーラインに注意しなければならず、肩越しカットなどの手法を使うことでそのショックを緩和する。
  • 銀レフ（レフ板） : 光の反射板。屋外撮影において、電源や機材の不備により照明をたくことができない場合に重宝する。太陽光を反射させて「おさえ」のライトにするもの。
  • クランクアップ・クランクイン : 全ての撮影を終了すること（アップ）。最初の撮影を開始すること（イン）。自然とインのほうがアップより多く経験する。
  • クレジット : 映像の最初に出てくる、制作会社やタイトル、ディレクターなどの名称を示したキャプションやフリップ。あるいは、最後に出てくるスタッフロールを指すこともある。
  • クレーン : 撮影用のクレーンのこと。平面上を移動するドリーをさらに進化させたもので、高さを含めた３軸方向の移動を自在に扱える。プログラム制御のものも多い。moveで自作して失敗した過去がある。
  • クローズアップ : 大写しのこと。接写。コンテには「C-UP」と書く。「クローズアップ現代」にちなんで「クローズアップ変態」というPVを作りたかったことがある。
  • クロスフェード : オーバーラップと同義語。動画と動画、音声と音声を重ねて、ゲインを変化させるトランジション。場合によってはディゾルブとも。
  • クロマキー : 画面合成の手法の一つ。ブルーまたはグリーンのバックを用いて撮影された映像に、特定色の部分のみの抜き出しを行うことで合成を実現する。
  • 黒み : 真っ黒な画面のこと。黒みを入れる、黒をおく、などといった使い方をする。どんな動画であれ、最初の数秒を黒みか白みで始めることが慣習。
  • ゲイン : アンプの増幅機能のこと。音響用語。
  • ゲリラ : 撮影許可のいる場所で無許可撮影を行うこと。いざとなったら「慶応義塾大学で映画を研究しているものですが・・・」で乗り切ろう。
  • 構成台本 : シナリオを作成する前段階で、シーン構成だけ記述した台本。絵コンテに移す前の脚本を作る前のシナリオを書く前のおおざっぱな構成を書いたもの。
  • 香盤表 : 影をスムーズに行なうために、登場人物や小道具、衣装等を表にしたもの。　シーンごとに分類したものや登場人物により分類したものなどがあり、目的に応じて必要な香盤表を作成する。
  • ごしショット（○○越し） : 何かをフレームの手前において、奥のものをフォーカスして撮影する手法。たとえば肩越しショットなど。
  • コマ撮り : 一フレームずつ撮影する方法。ストップ・モーションとも。
  • コントラスト（明度) : 明度のこと。
  • サイレント : 無声映画のこと。あるいは、録音が必要ないシーンのこと。BGMをかぶせたり、アフレコが確定している場合のこと。だからといって現場ではしゃぐと編集者に嫌われる。
  • 残留 : 編集単位。「〜のシーンは３残（留）でいける」などといった使い方をする。
  • シナリオ : 脚本。ト書きとセリフによって構成される。これに字コンテや絵コンテが加わったものをコンテ書きとかシナリオコンテと言ったり、ト書きを少なめにしてセリフに特化したものを台本と呼んだりする。
  • 絞る : レンズの露光を少なくすること。ライトのフォーカスを少なくすること。音量を下げること。など、何かを下方調節すること。
  • 字幕 : テロップとほぼ同義語。キャプションとも。素人とプロの違いがかなり明確に出る部分。
  • ジャンプカット : 同じ構図で撮影された映像の中抜きをすること。インタビュー映像などをイメージするとわかりやすい。映画で使うとショックのある演出になる。
  • 順撮り : コンテどおり、カット割りどおりの順番で撮影すること。効率が悪いが、スタッフにも完成の画がイメージできるという利点がある。
  • スクリプター : 撮影にかかわるデータ、たとえば「○シーン○カットの○テイクは、照明の具合が微妙だったのでキープだが、○シーンの状況によっては〜」などあらゆるデータを記録する係。助監督とも。
  • 捨てカット : 余分なカット。予定された尺や演技の幅より多めにカメラをまわしておくことで編集の負担を軽減するカット。「使わないかもしれないけれど一応撮っておこう」カットのこと。これは時間が許す限りあると余裕が出る。
  • センチュリースタンド : 自在アームの名称。　スタンド部と「槍」と呼ばれるアーム部をユニバーサルジョイントで接合して様々な形状にする事ができる。主に照明用に使用され、カポックを取り付けたり、光拡散用のパラフィン紙を貼ったりあらゆる事に利用できる。MOVEの今年度の部費で購入する機材の筆頭。
  • つなぐ、つながる : 編集して不自然にならないということ。この感覚については別記。
  • DV（ディーブイ) : 6.36ｍｍ幅のテープを使った民生用小型デジタルVTRシステムの名称。　DVキャメラやテープそのものを指す場合もある。AVカウンターでは「DVX」というカメラがこれにあたり、長年にわたりMOVEの主力装備であった。
  • ディフューザー : 照明の光が強すぎたり、色味が違うときに、拡散させるためのシート。トレペでも同様の効果が得られるが、専用の機材もある。
  • ティルト : カメラを上下に振ること。左右に振ることをパンという。ティルト・アップやティルト・ダウンとよぶ。パンアップ、パンダウンとも言う。

絵コンテと演出技法について

絵コンテとは、Continuity（連続した、一続きの）という言葉から来るもので、映像の設計図である。通常は右図のようなものをさす。絵コンテのほかにも、写真コンテや字コンテなどの手法がある。要するに、ディレクター（演出）である監督がシナリオから得た映像のイメージを、どうスタッフに伝達するかというところに主眼がおかれたモノである。これの良し悪しは、監督のセンス以上に映像の出来に左右することもあるほど重要である。


絵コンテの書き方

絵コンテの代表的な用紙は以下のようなものである。このシートのそれぞれのフレームには、「１カット」を書き、それぞれの内容を書き記すところは最低限のセリフと、パンやフォーカスなどの記号(P・Downなど)の撮影記号を書くことだけが共通の慣習で、あとはスタジオによりけり、スタッフによりけりで慣習や文化は違う。もちろん、ロケーションに撮影の前に行くことができ、役者の代わりにカメラに映ってくれる人がいるならば、写真コンテのほうが絵コンテより良いことは間違いない。画に間違いがなくなるからだ。

映像表現、演出技法

口うるさいようですが、「カットの感覚」というものが存在して、それはディレクターやスタジオによって違う文化ではあるものの、映像のクオリティにかかわるものです。純粋にカットの長さのこともそうだし、どのカットの次にどのようなカットが来るか、どのようなアングル・ポジションが来るか、といった文化のことです。

カットのショック

カットがショックを持つということがあります。というのは、視聴者が存在して、作り手が存在して、その両者のイメージする映像の流れが違うという状況のことを言います。たとえば、二人の人間が会話しているとして(右図)、このカットの次に来る構図は何であると考えるでしょうか。

  • どちらかの人の顔のアップ
  • 手のアップ
  • 鼻のアップ
  • 二人の真上から、俯瞰した映像

    • など、色々なカットが想像されえます。たとえば、分類するならば、手のアップなどはドキュメンタリー的、鼻のアップならコメディ的、俯瞰の映像なら討論バラエティ的、というような「それぞれの視聴者の雰囲気にあった自然なつながり」が存在します。

    じゃあ、映画ではどういうことか、というと、「カット割りの一貫性、演出の一貫性」というものが必要になってきます。視聴者と演出者のイメージの流れ、テンポ、空気感をあわせることが重要です。

    ショックのあるカットは、PVでは有効です。ジャンプカットやストップモーションなどの技法は、見るものを釘付けにして、次にどのようなカットが来るのかを期待させます。そういったショックのあるカットはプロモーションビデオなどパブリックな場で流れる作品では多用されるものです。

    しかし、そのショックを効果的に使うことが映画では求められます。安らぎや、自然な流れを表現したいシーンで、不要なショックを与えること（画面上の不自然や、つながりの不自然を見せてしまうこと）は、ひいては「素人の映像、レベルの低い映像」ということにつながります。

    とにかく視聴者と一体化することが重要です。カメラワークで視線誘導したり、小道具などを繰り返し使うことでイメージとして象徴づけたり、さまざまな演出手法があります。
